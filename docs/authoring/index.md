Jupyter Book inherits MyST Markdown's full authoring feature set.
The pages below cover the most common features, and the sections after that link to the MyST guide for deeper reference.

```{toc}
:context: children
```

## Typography

MyST extends CommonMark with footnotes, definition lists, subscripts and superscripts, smart-quotes, and other typographic features. See [](xref:guide/typography).

## Admonitions

Callout boxes for notes, warnings, tips, and other side content. Admonitions can be collapsed by default and support custom titles and classes. See [](xref:guide/admonitions).

## Figures

Embed images with captions, labels, alignment, and width controls. Figures can be cross-referenced from anywhere in your book. See [](xref:guide/figures).

## Tables

Author tables in Markdown or load them from external CSV/TSV files, with support for captions, labels, and column alignment. See [](xref:guide/tables).

## Code blocks

Syntax-highlighted code with line numbers, emphasized lines, captions, and filenames. See [](xref:guide/code).

## Dropdowns, cards, and tabs

Interactive content layouts for collapsible details, card grids, and tabbed panels. See [](xref:guide/dropdowns-cards-and-tabs).

## Diagrams

Render Mermaid diagrams (flowcharts, sequence diagrams, etc.) directly in your pages. See [](xref:guide/diagrams).

## Asides and marginalia

Place content in the page margin as a sidenote or aside. See [](xref:guide/asides).

## Blocks

Group content into logical blocks that share metadata, classes, or visibility settings. See [](xref:guide/blocks).

## Citations and bibliographies

Cite from a BibTeX file using `@key` syntax and render a formatted reference list. See [](xref:guide/citations).

## Cross-references

Label any element (figures, equations, sections, tables) and reference it from anywhere in your book. See [](xref:guide/cross-references) and the [tutorial](../get-started/cross-reference.md).

## External references

Link to Wikipedia, DOIs, GitHub source, RRIDs, and other MyST sites with rich previews on hover. See [](xref:guide/external-references).

## Embeds and reuse

Embed content from other pages or external sources to keep documentation DRY. See [](xref:guide/embed).

## Proofs and theorems

Numbered, cross-referenceable proofs, theorems, lemmas, and corollaries. See [](xref:guide/proofs-and-theorems).

## Exercises

Author exercises with optional solutions that can be hidden or shown. See [](xref:guide/exercises).

## Glossaries and terms

Define terms once and reference them throughout your book. See [](xref:guide/glossaries-and-terms).

## Writing in LaTeX

Embed raw LaTeX for formulas or content that benefits from the LaTeX ecosystem. See [](xref:guide/writing-in-latex).

## Inline options for classes and configuration

Configure directives and roles using compact inline option syntax. See [](xref:guide/inline-options).
