Most aspects of a Jupyter Book website are configured in `myst.yml` and the topics below link to the MyST guide for the full reference.

## Website templates

Choose and configure a site theme to control the look and feel of your website. See [](xref:guide/website-templates).

## Navigation

Customize the top navigation bar, sidebars, and site actions. See [](xref:guide/website-navigation).

## Site metadata

Configure site title, description, logo, and favicon. See [](xref:guide/website-metadata).

## Custom styles

Apply your own CSS to override theme defaults. See [](xref:guide/website-style).

## Landing pages

Build a custom landing page that introduces your project. See [](xref:guide/website-landing-pages).

## Site downloads

Offer downloadable assets (PDFs, data files, etc.) from your site. See [](xref:guide/website-downloads).

## SEO and social cards

Configure Open Graph metadata, social preview cards, and SEO-friendly markup. See [](xref:guide/seo-and-social).

## Analytics

Add Plausible or Google Analytics to track usage of your site. See [](xref:guide/analytics).

## Accessibility and performance

Performance tips and accessibility features for MyST sites. See [](xref:guide/accessibility-and-performance).
