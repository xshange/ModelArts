Jupyter Book can export your project to several static document formats (PDF, Word, JATS XML, etc) using the same content you build for the web.
For a step-by-step walkthrough, see [](xref:guide/quickstart-static-exports).

## PDF documents

Build publication-quality PDFs using Typst or LaTeX templates from a wide selection of journal and preprint formats. See the [PDF tutorial](../get-started/export-pdfs.md) and [](xref:guide/creating-pdf-documents).

## Word documents

Export your project to `.docx` for collaborators who use Microsoft Word. See [](xref:guide/creating-word-documents).

## JATS XML

Export to Journal Article Tag Suite XML for scholarly publishing workflows. See [](xref:guide/creating-jats-xml).

## CITATION.cff

Generate a `CITATION.cff` file from your project metadata so others can cite your work. See [](xref:guide/creating-citation-cff).

## Document parts

Templates can require named parts (abstract, acknowledgements, footer, sidebar, etc.) that map to sections of your document. See [](xref:guide/document-parts).

## Document exports overview

For a complete overview of MyST's export pipeline and supported output formats, see [](xref:guide/documents-exports).
